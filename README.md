# todolist




## About
Telegram bot organized as a task manager 
- pyTelegramBotAPI(telebot) to use Telegram API
- have consistent memory storage - MySQL


## Installation

### Telebot
```
$ pip install pyTelegramBotAPI
```

### MySQL connector 
``` 
python -m pip install mysql-connector-python
```
